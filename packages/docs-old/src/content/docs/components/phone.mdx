## API

### `Phone`

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| `children` | Content inside the phone screen | `React.ReactNode` | - |
| `color` | Custom frame color | `string` | - |
| `className` | Classes for phone container | `string` | - |
| `displayClassName` | Classes for display area | `string` | - |

## Accessibility

- Phone mockup is decorative - content inside should be accessible
- Images inside should have appropriate `alt` text
- Interactive elements inside remain fully accessible
