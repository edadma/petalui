## API

### `Join`

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| `direction` | Join direction | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `children` | Elements to join together | `React.ReactNode` | - |
| `className` | Additional CSS classes | `string` | - |
