## API

### `ConfigProvider`

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| `locale` | Locale configuration object | `Locale` | `enUS` |
| `componentSize` | Default size for all components | `'small' \| 'middle' \| 'large'` | `'middle'` |
| `direction` | Layout direction | `'ltr' \| 'rtl'` | `'ltr'` |
| `prefixCls` | Prefix for CSS classes | `string` | - |
| `getPopupContainer` | Default container for portals (modals, dropdowns) | `(triggerNode?: HTMLElement) => HTMLElement` | - |
| `children` | Child components | `React.ReactNode` | - |

## Available Locales

Import locales from `asterui/locale`:

```tsx
import { zhCN, jaJP, koKR, deDE, frFR, esES, ptBR, enGB, enCA } from 'asterui/locale'
```

| Locale | Language |
|--------|----------|
| `enUS` | English (US) - default |
| `enGB` | English (UK) |
| `enCA` | English (Canada) |
| `zhCN` | Chinese (Simplified) |
| `jaJP` | Japanese |
| `koKR` | Korean |
| `deDE` | German |
| `frFR` | French |
| `esES` | Spanish |
| `ptBR` | Portuguese (Brazil) |

## Hooks

### `useConfig`

Access the current configuration values:

```tsx
import { useConfig } from 'asterui'

function MyComponent() {
  const { locale, componentSize, direction } = useConfig()
  // ...
}
```

### `useLocale`

Access the current locale:

```tsx
import { useLocale } from 'asterui'

function MyComponent() {
  const locale = useLocale()
  // ...
}
```

## Accessibility

- Use `direction="rtl"` for right-to-left languages like Arabic and Hebrew
- Locale settings affect screen reader text in components like Empty, Pagination, etc.
- Nested providers allow different regions of your app to have different configurations
