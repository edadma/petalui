## API

### `ConfigProvider`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `locale` | Objeto de configuração de localização | `Locale` | `enUS` |
| `componentSize` | Tamanho padrão para todos os componentes | `'small' \| 'middle' \| 'large'` | `'middle'` |
| `direction` | Direção do layout | `'ltr' \| 'rtl'` | `'ltr'` |
| `prefixCls` | Prefixo para classes CSS | `string` | - |
| `getPopupContainer` | Contêiner padrão para portais (modais, dropdowns) | `(triggerNode?: HTMLElement) => HTMLElement` | - |
| `children` | Componentes filhos | `React.ReactNode` | - |

## Localizações Disponíveis

Importar localizações de `asterui/locale`:

```tsx
import { zhCN, jaJP, koKR, deDE, frFR, esES, ptBR, enGB, enCA } from 'asterui/locale'
```

| Localização | Idioma |
|--------|----------|
| `enUS` | Inglês (EUA) - padrão |
| `enGB` | Inglês (Reino Unido) |
| `enCA` | Inglês (Canadá) |
| `zhCN` | Chinês (Simplificado) |
| `jaJP` | Japonês |
| `koKR` | Coreano |
| `deDE` | Alemão |
| `frFR` | Francês |
| `esES` | Espanhol |
| `ptBR` | Português (Brasil) |

## Hooks

### `useConfig`

Acessar os valores de configuração atuais:

```tsx
import { useConfig } from 'asterui'

function MyComponent() {
  const { locale, componentSize, direction } = useConfig()
  // ...
}
```

### `useLocale`

Acessar a localização atual:

```tsx
import { useLocale } from 'asterui'

function MyComponent() {
  const locale = useLocale()
  // ...
}
```

## Acessibilidade

- Use `direction="rtl"` para idiomas da direita para esquerda como Árabe e Hebraico
- Configurações de localização afetam o texto do leitor de tela em componentes como Empty, Pagination, etc.
- Provedores aninhados permitem que diferentes regiões do seu aplicativo tenham configurações diferentes
