AsterUI uses DaisyUI's theming system, giving you access to 30+ built-in themes and the ability to create custom themes.

## Setting a Theme

Set the `data-theme` attribute on your `<html>` element:

```html
<html data-theme="light">
```

Or dynamically with JavaScript:

```tsx
document.documentElement.setAttribute('data-theme', 'dark')
```

## Theme Colors

Each theme provides these semantic colors:

| Color | Description |
|-------|-------------|
| `primary` | Main brand color |
| `secondary` | Secondary brand color |
| `accent` | Accent/highlight color |
| `neutral` | Neutral gray tones |
| `base-100/200/300` | Background colors |
| `info` | Information (blue) |
| `success` | Success (green) |
| `warning` | Warning (yellow) |
| `error` | Error (red) |

## Using Theme Colors

In components, use the `color` prop:

```tsx
<Button color="primary">Primary</Button>
<Button color="secondary">Secondary</Button>
<Button color="accent">Accent</Button>
```

In CSS, use the color variables:

```css
.my-element {
  background-color: oklch(var(--p)); /* primary */
  color: oklch(var(--pc)); /* primary-content */
}
```

Or with Tailwind classes:

```tsx
<div className="bg-primary text-primary-content">
  Themed content
</div>
```
