Buttons trigger actions or navigate to other pages. AsterUI buttons support multiple colors, variants, sizes, and states.

## Import

```tsx
import { Button } from 'asterui'
```

## API

### Button Props

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| `color` | Button color | `'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error' \| 'neutral' \| 'ghost'` | - |
| `variant` | Button style variant | `'solid' \| 'outline' \| 'soft' \| 'ghost' \| 'link'` | `'solid'` |
| `size` | Button size | `'xs' \| 'sm' \| 'md' \| 'lg'` | `'md'` |
| `shape` | Button shape | `'default' \| 'square' \| 'circle'` | `'default'` |
| `loading` | Show loading spinner | `boolean` | `false` |
| `disabled` | Disable the button | `boolean` | `false` |
| `block` | Full width button | `boolean` | `false` |
| `htmlType` | HTML button type | `'button' \| 'submit' \| 'reset'` | `'button'` |
| `onClick` | Click handler | `(e: MouseEvent) => void` | - |
| `className` | Additional CSS classes | `string` | - |

## Accessibility

- Buttons use native `<button>` elements for proper semantics
- Loading state adds `aria-busy="true"`
- Disabled buttons have `aria-disabled="true"`
- Focus states are visible for keyboard navigation
