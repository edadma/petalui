## API

### `Toggle`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `checked` | Estado marcado controlado | `boolean` | `-` |
| `defaultChecked` | Estado marcado predeterminado (no controlado) | `boolean` | `false` |
| `onChange` | Callback cuando cambia el estado del toggle | `(checked: boolean, event: ChangeEvent<HTMLInputElement>) => void` | `-` |
| `size` | Tamaño del toggle | `'xs' \| 'sm' \| 'md' \| 'lg'` | `'md'` |
| `color` | Color del toggle | `'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | `-` |
| `disabled` | Deshabilitar el toggle | `boolean` | `false` |
| `className` | Clases CSS adicionales | `string` | `-` |

## Accesibilidad

- Usa input checkbox nativo para accesibilidad
- Soporta activación por teclado con la tecla Space
- El estado marcado/desmarcado es anunciado por lectores de pantalla
- El estado deshabilitado se comunica correctamente
