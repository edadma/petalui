## API

### `Alert`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `children` | Contenido de la alerta | `ReactNode` | - |
| `type` | Variante de color de la alerta | `'info' \| 'success' \| 'warning' \| 'error'` | - |
| `closable` | Mostrar botón de cierre | `boolean \| { onClose?: () => void; closeIcon?: ReactNode; afterClose?: () => void }` | `false` |
| `outline` | Estilo de contorno | `boolean` | `false` |
| `dash` | Estilo de contorno punteado | `boolean` | `false` |
| `soft` | Estilo suave | `boolean` | `false` |
| `vertical` | Disposición vertical | `boolean` | `false` |
| `className` | Clases CSS adicionales | `string` | - |

## Accesibilidad

- Usa `role="alert"` para lectores de pantalla
- El color no es el único indicador - los iconos proporcionan contexto adicional
- Los botones de acción son accesibles mediante teclado
