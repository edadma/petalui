## API

### `Toggle`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `checked` | 受控勾选状态 | `boolean` | `-` |
| `defaultChecked` | 默认勾选状态(非受控) | `boolean` | `false` |
| `onChange` | 开关状态更改时的回调 | `(checked: boolean, event: ChangeEvent<HTMLInputElement>) => void` | `-` |
| `size` | 开关大小 | `'xs' \| 'sm' \| 'md' \| 'lg'` | `'md'` |
| `color` | 开关颜色 | `'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | `-` |
| `disabled` | 禁用开关 | `boolean` | `false` |
| `className` | 额外的 CSS 类 | `string` | `-` |

## 可访问性

- 使用原生复选框输入以获得可访问性
- 支持使用空格键进行键盘激活
- 勾选/未勾选状态由屏幕阅读器宣布
- 禁用状态得到正确传达
